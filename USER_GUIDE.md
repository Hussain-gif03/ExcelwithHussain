## How to Use the Platform

### Browsing Courses
1. Click "Courses" in the navigation menu
2. View all three course levels
3. Click "View Modules" to see course content
4. Each course shows:
   - Course level and description
   - Number of modules
   - Module titles and descriptions

### Taking Lessons
1. Select a course and click "View Modules"
2. Click on a module to see its lessons
3. Click "Start Lesson" to begin
4. Read through the lesson content carefully
5. Try the practice exercises
6. Click "Mark as Complete" when finished
7. Progress to the next lesson

### Taking Quizzes
1. After completing all lessons in a module, take the quiz
2. Click "Start Quiz" button
3. Answer all questions (8-10 per quiz)
4. Submit your answers
5. View your score immediately
6. Passing score: 70% or higher
7. Retake if needed to improve your score

## Tips for Success

### Study Tips
1. **Follow the sequence**: Start with Beginner, then Intermediate, then Advanced
2. **Practice regularly**: Apply what you learn in real Excel
3. **Take notes**: Write down important formulas and shortcuts
4. **Do the exercises**: Hands-on practice is essential
5. **Review before quizzes**: Go through lessons again if needed

### Quiz Tips
1. **Read carefully**: Understand what each question asks
2. **Take your time**: No time limit on quizzes
3. **Learn from mistakes**: Review incorrect answers
4. **Retake if needed**: You can retake quizzes to improve scores
5. **Apply knowledge**: Use Excel while studying
