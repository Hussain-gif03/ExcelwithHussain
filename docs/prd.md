\n## 2. Website Features\n
### 2.1 Course Structure
The courses are organized into three main levels with comprehensive Excel skills coverage:

#### 2.1.1 Beginner Level
- Excel interface and navigation
- Basic data entry and formatting
- Cell references (relative, absolute, mixed)
- Basic formulas (SUM, AVERAGE, COUNT, MAX, MIN)
- AutoFill and Flash Fill
- Sorting and filtering data
- Basic charts and graphs
- Page setup and printing
- Working with multiple worksheets
- Basic cell formatting (fonts, colors, borders, alignment)

#### 2.1.2 Intermediate Level
- Logical functions (IF, AND, OR, NOT, IFERROR)
- Lookup functions (VLOOKUP, HLOOKUP, INDEX, MATCH)
- Text functions (CONCATENATE, LEFT, RIGHT, MID, TRIM, UPPER, LOWER)
- Date and time functions (TODAY, NOW, DATE, DATEDIF)
- Conditional formatting
- Data validation and drop-down lists
- Named ranges\n- Advanced filtering and sorting
- Subtotals and grouping
- Working with tables
- Chart customization and advanced chart types
- Sparklines
\n#### 2.1.3 Advanced Level
- Advanced formulas (SUMIF, SUMIFS, COUNTIF, COUNTIFS, AVERAGEIF)\n- Array formulas\n- Pivot tables and pivot charts
- Power Query basics
- Data consolidation
- What-if analysis (Goal Seek, Scenario Manager, Data Tables)
- Macros and VBA basics
- Recording and editing macros
- Custom functions
- Advanced data visualization
- Dashboard creation
- Protecting worksheets and workbooks
- Collaboration and sharing features
- Excel shortcuts and productivity tips

### 2.2 Core Features
\n#### 2.2.1 Practice Exercises
Interactive exercises for students to apply what they've learned in each module.

#### 2.2.2 Quizzes
Assessment quizzes at the end of each module to test student understanding.

#### 2.2.3 Progress Tracking
A dashboard showing students their learning progress, completed modules, quiz scores, and overall completion percentage.

#### 2.2.4 Certificate System\nStudents receive a certificate upon:\n- Completing all lessons across all three levels
- Passing all quizzes with the required score\n
#### 2.2.5 Contact & Consultation\nA dedicated section featuring:
- Instructor contact information
- Consultation request form
- FAQ section

## 3. Design Style

### 3.1 Color Scheme\n- Primary color: Deep green (#2D5F3F) representing growth and learning
- Secondary color: Bright teal (#00A896) for interactive elements
- Accent color: Warm orange (#F4A261) for highlights and call-to-action buttons
- Background: Clean white (#FFFFFF) with light gray (#F8F9FA) sections

### 3.2 Visual Elements
- Modern card-based layout for course modules with subtle shadows (2px 8px rgba(0,0,0,0.1))
- Rounded corners (8px border-radius) for a friendly, approachable feel
- Progress bars with gradient fills to visualize learning advancement
- Clean, minimalist icons in line style for navigation and features
- Smooth hover transitions (0.3s ease) on interactive elements

### 3.3 Layout
- Grid-based layout for course modules (3 columns on desktop, responsive on mobile)
- Clear visual hierarchy with ample white space for easy readability
- Sticky navigation bar for quick access to different sections
- Dashboard layout with widgets displaying progress metrics and upcoming lessons
