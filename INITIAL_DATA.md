# Complete Excel Training Content

## Overview
The database has been pre-populated with **comprehensive Excel training content** covering beginner to advanced levels. This is a complete curriculum ready for students to learn Microsoft Excel.

## Complete Course Content

### Courses (3)
1. **Beginner Level** - Master the fundamentals of Microsoft Excel
2. **Intermediate Level** - Advance your Excel skills with complex formulas and data analysis
3. **Advanced Level** - Become an Excel expert with macros, pivot tables, and automation

### Modules (9 total - 3 per course)

#### Beginner Level Modules:
1. **Getting Started with Excel** - Interface, navigation, cell references, named ranges
2. **Basic Formulas and Functions** - Arithmetic operators, SUM, AVERAGE, COUNT, cell references
3. **Cell Formatting and Styles** - Number formatting, fonts, alignment, borders, colors

#### Intermediate Level Modules:
4. **Advanced Formulas** - IF functions, VLOOKUP, HLOOKUP, text functions
5. **Data Analysis Tools** - Sorting, filtering, data validation, conditional formatting
6. **Charts and Graphs** - Creating charts, formatting, advanced techniques, sparklines

#### Advanced Level Modules:
7. **Pivot Tables and Pivot Charts** - Creating pivot tables, grouping, calculations
8. **Macros and VBA Basics** - Recording macros, automation, macro security
9. **Advanced Data Visualization** - Interactive dashboards, slicers, KPIs
